Rest API has detected access from a browser and operates in "help" mode. You can use this mode to explore this API. If you wish to use Rest API, X-LlamaCpp-Client header must be set, optionally containing client identificator, for example: "restclient/myscript-0.0.1" To set User Agent header with curl: curl http://server:port/ -H "X-LlamaCpp-Client: restclient" To set User Agent header with XMLHttpRequest: request=new XMLHttpRequest(); request.setRequestHeader("X-LlamaCpp-Client", "restclient");
